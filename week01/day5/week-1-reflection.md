## Merge Conflict Lessons

- The conflict markers looked like `<<<<<<< HEAD`, `=======`, and
  `>>>>>>> feature/about-update`, which showed the two different versions of the
  same line coming from main and the feature branch.

- I chose to keep the version from main because it already matched the direction
  I wanted for the project.

- Next time, I would try to communicate changes better before working on the
  same line in different branches to reduce conflicts.
