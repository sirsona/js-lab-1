# Day 2 Journey

When someone type a URL into a browser and presses Enter, the browser read the
URL to know which website to open. It then ask DNS to find the IP address of
that website.

Next, the browser send an HTTP request to the server using that IP address. The
server receives the request and sends back the webpage data.

The browser then reads the HTML and shows the webpage.

The internet is the global network of connected computer, while the world wide web is a system of websites and web pages that runs on top the internet.
